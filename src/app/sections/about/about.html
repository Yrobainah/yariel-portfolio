Sobre mí

¿Quién soy?

Soy desarrollador Backend especializado en Java y Spring Boot, con formación en Ingeniería Informática y experiencia construyendo aplicaciones web completas. Creo software limpio, mantenible y orientado a resolver problemas reales.

¿Qué construyo?

Desarrollo APIs REST con Spring Boot, Spring Security, JWT, PostgreSQL y MySQL, además de interfaces con Angular, Vue 3 y Tailwind CSS. En SmartInventory integré autenticación, roles, dashboard operativo, Docker Compose, arquitectura por capas y una API documentada.

¿Cómo trabajo?

Voy más allá de un simple CRUD: busco una arquitectura sólida, interfaces cuidadas y una experiencia de usuario profesional, con atención a la calidad del código, la documentación y las buenas prácticas de desarrollo.
